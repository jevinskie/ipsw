## ipsw appstore

Interact with the App Store Connect API

```
ipsw appstore [flags]
```

### Options

```
  -h, --help         help for appstore
  -i, --iss string   Issuer ID
  -j, --jwt string   JWT api key
  -k, --kid string   Key ID
  -p, --p8 string    Path to App Store Connect API Key (.p8)
```
